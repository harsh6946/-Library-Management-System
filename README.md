# 📚 Library Management System

A simple **Python + MySQL** project to manage books in a library.  
You can add and view books using a database.

## 🚀 Features
- Add new books
- View all books
- Uses MySQL for storage

## 🛠️ Tech Stack
- Python
- MySQL
